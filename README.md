City-of-Bloomington.mysql
=========
Ansible role for MySQL

Requirements
------------

City-of-Bloomington.linux

Example Playbook
----------------

```yml
- hosts: linux-solr
  become: yes
  roles:
  - City-of-Bloomington.linux
  - City-of-Bloomington.mysql
```
